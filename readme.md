# Grafite Documentation

This is the Grafite package documentation repository. It uses Mkdocs.

## Packages

- Builder
- CMS
- Commerce (in progress)
- CLI (in progress)
- CrudMaker
- FormMaker
- HomeForge
- Crypto

## Install

```
pip install mkdocs
pip install mkdocs-bootswatch
```

## Commands

```
mkdocs serve
mkdocs build
mkdocs build --clean // overwrites the current
```
