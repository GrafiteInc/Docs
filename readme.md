# Grafite Documentation

This is the Grafite package documentation repository. It uses Mkdocs.

## Active Packages

- Scaffold
- FormMaker
- Database
- Helpers

## Archived Packages

- Builder
- CMS
- Commerce
- CLI
- CrudMaker
- HomeForge
- Crypto

## Install

```
pip install mkdocs
pip install mkdocs-bootswatch
```

## Commands

```
mkdocs serve
mkdocs build
mkdocs build --clean // overwrites the current
```
